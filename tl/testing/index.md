# Pagsusuri ng Dysphagia — Mga Screening Tool sa Tagalog

Ang seksyong ito ay nagbibigay ng mga simpleng screening tool na maaaring gamitin ng mga tagapag-alaga upang matukoy kung mayroon o lumalala ang dysphagia ng kanilang employer. Ang mga tool na ito ay hindi kapalit ng diagnosis ng doktor, ngunit makakatulong sa inyo na malaman kung kailan kailangan ng propesyonal na ebalwasyon.
