# 居家吞嚥訓練計劃：家屬指導與每日20分鐘訓練組合

## 概覽

吞嚥障礙的康復不僅在醫院或診所進行——大量實證顯示，居家堅持訓練是維持及改善吞嚥功能的關鍵。對香港許多患者而言，公立醫院言語治療（SLT）複診間距可長達數週至數月，居家訓練填補了這個空白。本指南適合家屬在言語治療師指導後，協助患者在家執行。

**重要前提**：本訓練計劃須由言語治療師（SLT）首先評估患者，確認各項練習適合該患者，並根據患者具體情況調整。家屬不應在未獲SLT授權下自行開始或調整訓練強度。

---

## 第一部分：三項核心訓練介紹

### 訓練1：Shaker 運動（頭部抬升運動）

**目標**：加強舌骨上肌群（suprahyoid muscles），改善環咽肌開放，減少食物在食道入口的殘留。

**操作步驟**：
1. 患者平躺在床或瑜伽墊上，雙肩保持貼地
2. 僅抬起頭部（不抬肩膀），眼睛望向自己的腳趾
3. 有兩個模式：
   - **等長模式（Isometric）**：維持頭部抬起姿勢60秒，休息60秒，重複3次
   - **等張模式（Isotonic）**：慢慢抬頭再放下，重複30次（每次約1–2秒）
4. 同一節訓練中，等長與等張各做一輪

**家屬注意事項**：
- 患者若有頸椎問題（如椎管狹窄、頸椎骨折史），訓練前須先諮詢醫生
- 部分患者初期頸部肌肉酸痛屬正常，若出現頸部疼痛或頭暈應立即停止
- 宜在餐前30–60分鐘進行（避免剛進食後做此運動）

---

### 訓練2：舌肌力量訓練（Tongue Strengthening）

**目標**：加強舌頭頂壓力量，改善食團推進及咽喉期清除效率。

**操作步驟（無設備版本）**：
1. **舌頂上顎（Palatal Push）**：舌頭用力頂住上顎最高位置，維持5秒，重複10次
2. **舌左右推牙**：舌尖用力頂左側臉頰，SLT 或家屬輕輕從外側施加阻力，維持3秒；左右各10次
3. **舌伸長伸縮**：舌頭盡力向前伸出，維持2秒，再縮回，重複10次（訓練舌頭靈活性）
4. **舌根後縮（Tongue Base Retraction）**：發「嘎」（ga）音或做誇張的吞嚥動作，感受舌根後縮，重複15次

---

### 訓練3：Mendelsohn 手法

**目標**：延長喉部抬升時間，增加環咽肌開放持續時間，改善咽喉期食團通過。

**操作步驟**：
1. 患者先做一次普通吞嚥，感受喉結（adam's apple）上下移動的感覺
2. 再次吞嚥時，當喉結抬到最高點時，**用力維持在高位3–5秒**，不讓喉結立即下降
3. 維持成功後，放鬆讓喉結自然回落
4. 重複10次，組間休息30秒

**家屬輔助技巧**：
- 可輕輕放手指在患者喉結側面，幫助患者感受喉結的位置和移動
- 初學者常見錯誤：憋氣代替真正的喉部維持——提醒患者「維持喉結高位，而非憋住氣」
- 此手法難度較高，部分患者需較長時間掌握，請耐心練習

## 第二部分：每日20分鐘訓練時間表

以下為建議的每日訓練組合，總時間約20分鐘（含休息）：

| 時段 | 項目 | 時間 |
|------|------|------|
| 熱身（可選） | 口唇運動：鼓腮→嘟嘴→露齒，各5次 | 2分鐘 |
| 第1組 | Shaker 等長（60秒維持×3，組間60秒休息） | 6分鐘 |
| 休息 | 喝水（若SLT允許）或休息 | 1分鐘 |
| 第2組 | Shaker 等張（30次慢速） | 2分鐘 |
| 休息 | — | 1分鐘 |
| 第3組 | 舌肌訓練（4個動作各1輪） | 4分鐘 |
| 休息 | — | 1分鐘 |
| 第4組 | Mendelsohn 手法（10次） | 3分鐘 |

**建議每週訓練5天，休息2天**（讓肌肉有充分恢復時間）。

## 第三部分：訓練卡設計與記錄追蹤

### 製作家用訓練卡

SLT 複診後，請要求治療師為患者製作個人化訓練卡，或參考以下要素自製：

**訓練卡必備內容**：
- 患者姓名及訓練開始日期
- 每項訓練的清晰圖示（可請治療師提供或從衛生署/醫管局資源下載）
- 每項訓練的重複次數、維持時間、組數
- 「可以做」和「不可以做」（如Shaker是否有頸椎限制）
- 緊急情況：若患者出現嗆咳明顯加重、呼吸困難，立即停止訓練並聯絡醫護

**每日記錄表格**（簡單版）：

| 日期 | Shaker完成 | 舌肌完成 | Mendelsohn完成 | 備註（有否嗆咳、疲勞） |
|------|-----------|---------|---------------|----------------------|
| | ✓/✗ | ✓/✗ | ✓/✗ | |

將記錄帶到下次SLT覆診，幫助治療師評估進展。

## 第四部分：定期SLT覆診安排

### 覆診頻率建議

| 患者狀態 | 建議覆診間距 |
|----------|-------------|
| 急性腦卒中後（住院期） | 每1–3天評估一次 |
| 出院後急性復康期 | 每2–4週覆診 |
| 穩定居家訓練期 | 每4–8週覆診 |
| 維持期（功能穩定） | 每3–6個月覆診或按需要 |

### 香港覆診安排的實際建議

- **醫管局門診**：覆診等候時間長，建議出院時主動詢問 SLT 下次覆診日期，並確認緊急聯絡安排（若吞嚥功能突然惡化的處理方法）
- **私人 SLT 診所**：彈性較大，可按需加密覆診；費用每節約港幣600–1,200
- **電話/視像覆診**：部分 SLT 診所（包括部分醫管局服務）提供遙距評估，適合行動不便患者

### 需要提早覆診的警號

若出現以下情況，不應等至原定覆診，應盡快聯絡 SLT 或前往急症室：
- 嗆咳頻率或嚴重程度明顯增加
- 進食或飲水後出現發燒、咳嗽加劇（可能為吸入性肺炎先兆）
- 患者突然拒絕進食或明顯體重下降
- 聲音突然變嘶啞或濕潤感加重

---

## 總結

居家吞嚥訓練是醫院治療的延伸，Shaker 運動、舌肌訓練和 Mendelsohn 手法的每日20分鐘組合，有助維持及改善吞嚥功能。家屬的支持和記錄追蹤，以及定期 SLT 覆診，是確保居家訓練安全有效的關鍵。

**免責聲明**：本文所述訓練須在言語治療師評估及授權後方可執行，不應自行開始或調整。
