# T/SATA 094-2025大湾区吞咽障碍食物质地团体标准完整解读

## 一、标准背景：大湾区卫生融合的现实需要

2025年正式发布的《T/SATA 094-2025 大湾区吞咽障碍食物质地标准》是中国大陆首个系统对接IDDSI国际框架的吞咽饮食团体标准，由中国标准化协会（SATA）批准，适用范围覆盖粤港澳大湾区的医疗、养老及餐饮服务领域。

### 为何需要大湾区专项标准？

粤港澳大湾区的医疗融合面临独特的挑战：
- **香港**已全面推行IDDSI框架，公立医院、养老机构普遍使用
- **内地**（广东省为主）沿用传统流食/半流食/软食分类，体系不兼容
- 老年人跨境养老、跨机构转诊日益频繁，质地等级无法互认导致衔接断裂

T/SATA 094-2025正是为解决这一互认问题而生，通过在内地建立IDDSI兼容的本地化标准，降低大湾区内跨机构、跨境转诊时的信息断层风险。

## 二、核心条款详解

### 2.1 七大质地等级定义

T/SATA 094-2025采纳IDDSI 0-6级分类（不含IDDSI 7级普通食物，因该等级无需特别规范），并附加大湾区适用的中文等级名称：

| T/SATA等级 | 对应IDDSI等级 | 中文名称 | 核心特征 |
|------------|-------------|---------|---------|
| 质地0级 | IDDSI 0 | 稀流质 | 流速≈水，10秒注射器剩余≤1ml |
| 质地1级 | IDDSI 1 | 微稠流质 | 轻微增稠，流速略慢于水 |
| 质地2级 | IDDSI 2 | 低稠流质 | 明显增稠，从汤匙缓慢滴落 |
| 质地3级 | IDDSI 3 | 中稠/液态泥状 | 无颗粒，可用杯饮用或汤匙进食 |
| 质地4级 | IDDSI 4 | 高稠/泥状 | 光滑均匀，汤匙倾斜不滴落，可塑形 |
| 质地5级 | IDDSI 5 | 细碎湿润 | 颗粒≤4mm，充分湿润有黏聚性 |
| 质地6级 | IDDSI 6 | 软烂易嚼 | 颗粒≤15mm，舌压可碎 |

### 2.2 营养密度最低要求（T/SATA新增条款）

这是T/SATA 094-2025对IDDSI框架的重要补充。IDDSI本身不涉及营养密度，但T/SATA认识到质地越低，患者往往摄入量越少，须通过提高营养密度补偿。

| 质地等级 | 每100ml最低热量 | 每100ml最低蛋白质 |
|---------|--------------|----------------|
| 质地3级（液态） | ≥50kcal | ≥1.5g |
| 质地4级（泥状） | ≥80kcal | ≥3.0g |
| 质地5级（细碎） | ≥100kcal | ≥4.0g |
| 质地6级（软烂） | ≥120kcal | ≥5.0g |

**注意：** 上述为推荐最低值，并非强制要求。但机构申请认证时需提交营养密度检测数据作为符合性证明。

### 2.3 测试方法规范

标准认可以下测试工具：
1. **IDDSI标准注射器测试**：10ml一次性医用注射器，符合IDDSI操作规程
2. **叉子压力测试**：参照IDDSI方法，明确认可国内标准餐叉作为替代工具
3. **汤匙倾斜测试**：参照IDDSI方法
4. **黏度计测试（机构可选）**：使用旋转黏度计（如博勒飞DV系列）提供精确黏度值，适用于食品生产企业

## 三、与IDDSI的对照及差异分析

| 比较项目 | IDDSI | T/SATA 094-2025 |
|---------|-------|----------------|
| 发布机构 | 国际跨学科委员会 | 中国标准化协会（SATA） |
| 等级数量 | 8级（0-7） | 7级（0-6，不含7级） |
| 营养密度要求 | 无 | 有最低推荐值 |
| 中式食材参考 | 无 | 有（粥、豆腐、蒸蛋等） |
| 测试工具 | 规定叉子/注射器 | 兼容国内常见注射器及餐具 |
| 语言 | 英文为主，40+翻译版 | 简体中文为主，附繁体对照 |
| 法律效力 | 国际参考标准 | 中国团体标准（T标准），非强制 |

**关键对接点：** T/SATA与IDDSI的等级定义高度兼容，机构在文件中标注T/SATA质地等级时，可直接对应IDDSI等级使用，无需另行转换。

## 四、适用范围

T/SATA 094-2025明确规定以下场景均适用：

- **医疗机构**：住院部、康复科、神经内科、老年科
- **养老机构**：护理型床位、失能老人照护中心、长护险定点机构
- **社区服务**：日间照料中心、居家护理服务提供者
- **餐饮服务商**：为医疗/养老机构提供配餐外包服务的企业
- **食品生产企业**：生产并销售吞咽饮食产品的食品厂商

## 五、内地与香港的实施差异

### 语言差异

| 差异维度 | 香港（粤语环境） | 内地（普通话环境） |
|---------|--------------|----------------|
| 级别称呼 | "糊餐"（4级）、"剁碎餐"（5级） | "泥状食物"、"细碎湿润食物" |
| 测试指令语 | 粤语表述 | 普通话表述 |
| 患者教育材料 | 繁体中文 | 简体中文 |

### 食材差异

| 食材类别 | 香港常见食材 | 内地适配调整 |
|---------|------------|------------|
| 粥品 | 广式靓粥（煲2-3小时） | 普通白粥（煲1小时），需延长熬制时间或使用高压锅 |
| 豆腐 | 日式绢豆腐（质地接近4级） | 内酯豆腐/南豆腐，质地因品牌差异较大，须测试 |
| 面食 | 米粉、肠粉 | 龙须面、疙瘩汤（质地需专门控制） |

### 操作规范差异

香港养老机构普遍配备专职言语治疗师，吞咽评估作为入住常规评估项目；内地大多数基层养老机构无专职言语治疗师，T/SATA 094-2025认可经培训的护士执行标准化筛查工具（如柑橘测试），并在疑似吞咽障碍时转介专业评估。

## 六、认证路径

### 机构如何申请T/SATA 094-2025认证符合性声明

目前T/SATA 094-2025为团体标准，非强制认证标准，机构可通过以下方式证明符合性：

**路径A：自我声明**
1. 参照标准全文完成内部合规审查
2. 建立质地测试记录体系
3. 在机构宣传材料中标注"符合T/SATA 094-2025标准"

**路径B：第三方认证（推荐）**
1. 联系中国标准化协会（SATA）授权检测实验室
2. 提交机构质地管理文件及食品样本
3. 经实验室检测及现场审查后获颁符合性证书
4. 证书有效期：3年，期间须接受年度监督审查

**SATA合作实验室（参考）：**
- 华测检测（CTI）——广州、深圳均有食品实验室
- 国家食品质量安全监督检验中心（北京/广州）
- 广东省食品检验所

## 七、康樂齡（SeniorDeli）作为参编单位的背景

康樂齡（SeniorDeli）是T/SATA 094-2025的主要参编单位之一，在标准制定过程中贡献了：
- 大湾区吞咽饮食市场实践数据（患者质地需求分布、产品质地测试数据）
- 中式吞咽饮食食材数据库（覆盖粤式、潮式、港式常见食材的质地测试结果）
- 营养密度最低要求条款的草案建议（基于SeniorDeli产品研发数据）

康樂齡作为大湾区吞咽友好食品的领先品牌，其产品线按照T/SATA 094-2025和IDDSI双重标准设计生产，在香港、广州、深圳、澳门的养老机构及医院均有配送服务。

## 八、常见问题

**Q：T/SATA 094-2025是否具有法律强制效力？**
A：否。作为团体标准（T标准），其执行基于自愿原则。但长护险定点机构或政府采购招标中，可能将符合T/SATA标准列为加分项或入围条件。

**Q：内地养老机构是否必须按此标准运营？**
A：目前不强制，但按此标准建立的质地管理体系有助于机构通过等级评定审查，并降低法律风险。

**Q：如何获取T/SATA 094-2025标准全文？**
A：通过中国标准化协会官网（www.china-cas.org）或全国标准信息公共服务平台（std.samr.gov.cn）查询下载，部分条款提供免费预览，完整版需付费购买。
