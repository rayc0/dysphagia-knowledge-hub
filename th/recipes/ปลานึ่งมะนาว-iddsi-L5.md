# ปลานึ่งมะนาว — IDDSI L5 (Minced and Moist)

> **ระดับ IDDSI:** L5 — Minced and Moist
> **เวลาเตรียม:** 10 นาที | **เวลาปรุง:** 12–15 นาที
> **โปรตีน:** ~25 กรัม/มื้อ | **พลังงาน:** ~180 กิโลแคลอรี

---

## ปลานึ่งมะนาวในวัฒนธรรมไทย

ปลานึ่งมะนาวเป็นเมนูปลาที่ได้รับความนิยมสูงสุดชนิดหนึ่งในไทย โดยเฉพาะตามร้านอาหารทะเลและภัตตาคาร รสเปรี้ยวจากมะนาว กลิ่นหอมกระเทียมและพริก ทำให้ปลามีรสชาติจัดจ้าน

สำหรับผู้ป่วยกลืนลำบากระดับ L5 ปลากะพงนึ่งเป็นตัวเลือกที่ดีเยี่ยม เพราะ:
- เนื้อปลากะพงนุ่มตามธรรมชาติ แยกเป็นเส้น ๆ ได้ง่าย
- สับเป็นชิ้นเล็ก ≤ 4 มม. ได้โดยไม่ต้องเพิ่มขั้นตอน
- ชุ่มน้ำซอสตามธรรมชาติจากการนึ่ง
- ปรับรสซอสได้ตามความชอบโดยไม่กระทบเนื้อสัมผัส

---

## ส่วนผสม (สำหรับ 1 มื้อ)

### ปลา
| ส่วนผสม | ปริมาณ | หมายเหตุ |
|---|---|---|
| ปลากะพงขาวสด (ฟิลเลต์) | 120–150 กรัม | ไม่มีก้าง ไม่มีหนัง |
| ขิงบาง ๆ | 3 แผ่น | วางบนปลาก่อนนึ่ง |
| ต้นหอม | 1–2 ต้น | วางรองปลาเพื่อกลิ่น (เอาออกก่อนเสิร์ฟ) |

### ซอสมะนาว (ปรับลดความเผ็ด)
| ส่วนผสม | ปริมาณ | หมายเหตุ |
|---|---|---|
| น้ำมะนาว | 2 ช้อนโต๊ะ | |
| น้ำปลา | 1.5 ช้อนโต๊ะ | |
| น้ำตาลทราย | 1 ช้อนชา | |
| กระเทียมบด | 1 กลีบ | บดละเอียดมาก |
| น้ำเปล่า | 2 ช้อนโต๊ะ | เจือจางซอสเพื่อไม่ให้เค็มเกิน |
| พริกสด (ตัวเลือก) | ½ เม็ด | สับละเอียดมาก หรือตัดทิ้งหากระคายเคืองง่าย |

## วิธีทำ

### ขั้นตอนที่ 1: ตรวจก้างปลา (สำคัญมาก)
1. ใช้นิ้วลูบตามเนื้อปลาอย่างระมัดระวัง ทุกทิศทาง
2. ใช้แหนบดึงก้างที่พบออก
3. ทำซ้ำ 2–3 รอบ ก้างปลาเป็นอันตรายสำคัญสำหรับผู้ป่วยกลืนลำบาก
4. หากซื้อฟิลเลต์สำเร็จรูป ยังต้องตรวจอีกครั้งเสมอ

### ขั้นตอนที่ 2: นึ่งปลา
1. วางต้นหอมบนตะแกรงนึ่ง วางปลาทับ
2. วางขิงบนเนื้อปลา
3. นึ่งด้วยไฟแรง ฝาปิดสนิท 10–12 นาที
4. ทดสอบความสุก: แทงด้วยส้อมที่จุดหนาที่สุดของปลา เนื้อปลาควรแตกออกเป็นแผ่น ๆ ง่าย ไม่มีส่วนโปร่งใส

### ขั้นตอนที่ 3: สับปลาให้ถูก L5
1. นำปลาสุกออกจากหม้อ ทิ้งขิงและต้นหอม
2. ใช้ส้อม 2 อัน ฉีกเนื้อปลาออกตามเส้นใย
3. สับด้วยมีดให้ชิ้นเล็กลง ≤ 4 มม.
4. **สำคัญ:** ขนาดชิ้น ≤ 4 มม. เป็นข้อกำหนด IDDSI L5 ที่เข้มงวด

### ขั้นตอนที่ 4: เตรียมซอสและเสิร์ฟ
1. ผสมน้ำมะนาว น้ำปลา น้ำตาล กระเทียมบด น้ำ และพริก (หากใส่)
2. คนให้เข้ากัน ชิมรส ปรับตามชอบ
3. ราดซอสบนเนื้อปลาที่สับแล้ว
4. **ต้องเสิร์ฟทันที** เพราะน้ำมะนาวทำให้เนื้อปลาเริ่มเปลี่ยนเนื้อสัมผัสหากทิ้งไว้นาน

---

## การทดสอบ IDDSI L5

**Fork Pressure Test:**
1. วางเนื้อปลาสับบนส้อม
2. ออกแรงกดเบา ๆ เทียบเท่าแรงลิ้น (~130 กรัม)
3. **ผ่าน L5:** เนื้อปลาแตกออกง่าย ไม่ต้องออกแรงมาก
4. เนื้อปลาต้องชุ่มชื้น ไม่แห้ง ไม่ติดกัน

**ตรวจขนาดชิ้น:**
- ใช้ไม้บรรทัดขนาดเล็กหรือวัดด้วยนิ้ว ชิ้นต้องเล็กกว่า 4 มม.

## คุณค่าทางโภชนาการ

| สารอาหาร | ปริมาณ (120 ก.) |
|---|---|
| พลังงาน | 170–190 กิโลแคลอรี |
| โปรตีน | 24–28 กรัม |
| คาร์โบไฮเดรต | 5–7 กรัม (จากซอส) |
| ไขมัน | 3–4 กรัม |
| โอเมก้า-3 | ~400 มก. |
| วิตามิน D | ~200–300 IU |
| ฟอสฟอรัส | ~250 มก. |

ปลากะพงเป็นแหล่งโปรตีนสมบูรณ์ที่ดีที่สุดชนิดหนึ่งสำหรับผู้ป่วยกลืนลำบาก ให้โปรตีนสูงในปริมาณน้อย ไขมันต่ำ และย่อยง่ายกว่าเนื้อสัตว์บก

## ตัวเลือกปลาอื่นที่เหมาะกับ L5

หากหาปลากะพงไม่ได้หรือราคาแพง:

| ปลา | ข้อดี | ข้อระวัง |
|---|---|---|
| ปลาทับทิม | ราคาถูก เนื้อนุ่มดี | ก้างแฉลบมาก ต้องตรวจรอบคอบ |
| ปลานิล | หาง่าย ราคาถูก | คล้ายปลาทับทิม |
| ปลาแซลมอน | โอเมก้า-3 สูง เนื้อนุ่ม | ราคาสูงกว่า |
| ปลาดอรี่ (John Dory) | ก้างน้อย เนื้อขาว | แช่แข็งมักมีให้เลือก |

**หลีกเลี่ยง:** ปลาตัวเล็ก (ปลาดุก ปลาช่อน) ก้างเล็กมากเกินกว่าจะตรวจหมด

## เวอร์ชันสำหรับผู้ที่ต้องลดโซเดียม

- แทนน้ำปลาด้วยซีอิ๊วขาวสูตรโซเดียมต่ำ ½ ปริมาณ
- เพิ่มน้ำมะนาวแทนเพื่อชดเชยรสเค็ม
- ตัดพริกออกและใช้กระเทียมน้อยลง
