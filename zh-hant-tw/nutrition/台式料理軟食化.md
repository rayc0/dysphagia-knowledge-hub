# 台灣傳統料理的IDDSI質地改良完全指南

## 為什麼台式料理需要特別改良

台灣的飲食文化豐富多元，對於吞嚥障礙患者而言，保有熟悉的家鄉味不僅攸關營養攝取，更是心理健康與生活品質的重要支柱。然而，許多傳統台式料理含有複合質地（如脆皮、帶骨、黏稠湯汁混合固體），若未妥善改良，將大幅增加誤吸風險。

本指南以**IDDSI（國際吞嚥障礙飲食標準）**框架為基礎，提供台灣家庭最常見料理的質地改良方法。

## IDDSI質地等級快速對照

| 等級 | 名稱 | 特徵 |
|-----|-----|-----|
| L3 | 細碎及濕潤 | 4mm以下顆粒，需少量咀嚼 |
| L4 | 攪爛 | 可用舌頭壓碎，不需咀嚼 |
| L5 | 軟質及細碎 | 需輕度咀嚼，可用叉子壓碎 |
| L6 | 軟質及細碎（需咀嚼） | 一般軟食，需適度咀嚼 |
| L7 | 一般飲食 | 無限制 |

## 滷肉飯改良（IDDSI L4-L5）

滷肉飯是台灣最具代表性的家常料理，傳統版本含有切丁豬皮及米飯顆粒，對吞嚥障礙患者挑戰較大。

**L5改良版（軟爛五花肉+軟飯）：**
1. 五花肉選用肥瘦比例3:7，燉煮至少2小時至可輕易壓碎
2. 去除豬皮（膠質黏稠，易導致食物凝結成塊）
3. 白飯以米水比例1:1.5煮成軟飯，或使用米漿機打成米糊
4. 滷汁過濾去除八角、蔥段等固體香料，增稠至IDDSI L2濃稠度後淋上

**L4改良版（肉糊飯糊）：**
1. 滷肉加少量高湯以食物調理機打成均勻肉泥
2. 米飯加水打成米糊（稀稠度依患者需求調整）
3. 混合後用花型模具塑形，保留視覺吸引力

## 虱目魚料理改良

虱目魚刺多，需特別注意去骨問題。

**蒸虱目魚肚（IDDSI L5-L6）：**
1. 選用**去骨虱目魚肚**（各大超市均有販售，整塊去刺）
2. 以醬油、薑絲、米酒清蒸8至10分鐘
3. 蒸熟後以叉子確認可輕易壓碎，魚肉呈片狀分離即可

**虱目魚粥（IDDSI L3-L4）：**
1. 白粥以米水比例1:10煮成，粥底呈細膩糊狀
2. 去骨魚肉撥成細絲，加入滾粥中燙熟
3. 確認無魚刺殘留，加入薑絲增香

## 蚵仔麵線改良（IDDSI L3-L4）

傳統蚵仔麵線的蚵仔質地彈滑、麵線黏稠，整體質地複雜。

**去蚵改良版（IDDSI L4）：**
1. 去除蚵仔（蚵仔即使煮熟仍具彈性，不適合L4以下患者）
2. 麵線選用細麵線，燙軟後以剪刀剪成3公分以下小段
3. 芡汁以太白粉水調整至IDDSI L3濃稠度
4. 可加入豆腐丁或蛋花增加蛋白質

## 控肉飯（爌肉）改良（IDDSI L5）

爌肉（又稱控肉）以長時間燉煮聞名，質地相對容易改良。

**IDDSI L5改良：**
1. 豬腿肉（梅花肉）以醬油、冰糖、滷包燉煮3至4小時
2. 成品應可用筷子輕鬆穿透
3. 去皮後切片，確認每片厚度不超過1.5公分
4. 搭配滷汁泡軟的白飯或軟飯

## 台灣小吃攤選擇建議

外出用餐時，以下台灣小吃相對適合吞嚥障礙患者（需告知攤主特殊需求）：

- **清粥小菜攤**：選擇爛粥、豆腐、蒸蛋、魚鬆
- **台式自助餐**：選擇燉豆腐、蒸魚、炒軟嫩蔬菜，避免油炸物
- **麵攤**：選擇陽春麵（軟煮）或米粉湯，避免乾麵
- **早餐店**：蒸蛋、豆腐腦、稀飯優於饅頭、燒餅

避免：珍珠奶茶（珍珠彈性高，誤吸風險極高）、碗粿（黏稠易附著咽喉）、麻糬類。

## 超市採買IDDSI食材清單

以下台灣各大超市（全聯、家樂福、頂好）常見的適合食材：

**蛋白質來源（L4-L5）：**
- 嫩豆腐、雞蛋、去骨魚排、雞胸肉排
- 亞培安素、桂格完膳（現成液態配方）

**澱粉主食（L4-L5）：**
- 日本越光米（煮成軟飯易成型）
- 即食燕麥（加水泡開可達L4）
- 地瓜泥（市售冷凍地瓜泥）

**調味增稠：**
- 太白粉、葛粉（天然增稠）
- 桂格吞嚥輔助粉（專業凝固劑）
