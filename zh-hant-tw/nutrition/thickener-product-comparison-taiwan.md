# 台灣市售食物增稠劑比較：品牌選購指南

## 前言

增稠劑是吞嚥障礙患者日常照護中最常使用的輔助產品之一，卻也是照顧者最容易感到困惑的地方——市面上品牌繁多，各家成分不同，比例說明各異，使用後效果差距卻可以很大。

錯誤的增稠劑或使用方式，可能導致液體濃稠度不符合IDDSI標準，進而增加誤嗆風險。本文提供台灣市場的系統性比較，協助照顧者做出安全、合理的選擇。

---

## 第一部分：增稠劑的兩大類型

### 澱粉基增稠劑（Starch-Based）

**成分**：玉米澱粉、馬鈴薯澱粉、樹薯澱粉或其改良版本。

**優點**：
- 成本較低
- 口感較自然，不易產生異味
- 台灣超市及大型藥局均可取得基本版（如太白粉）

**缺點**：
- **濃稠度不穩定**：液體放置越久越稠，加熱後變稀
- **受唾液澱粉酶分解**：食物在口腔中咀嚼時，唾液會分解澱粉，導致濃稠度在口中下降
- **不透明**：使液體外觀變混濁
- 高溫飲品使用效果差

**適用情境**：預算有限、短期使用、室溫飲品、照顧者需要頻繁監測稠度變化。

---

### 膠基增稠劑（Gum-Based，主要為黃原膠 Xanthan Gum）

**成分**：黃原膠（xanthan gum）或關華豆膠（guar gum），或兩者混合。

**優點**：
- **高度穩定**：時間、溫度變化不顯著影響濃稠度
- **不受唾液澱粉酶影響**：在口腔中保持原有稠度
- 可用於冷熱飲品
- 通常透明，不影響飲品外觀
- 多項臨床研究顯示安全性較澱粉基更高

**缺點**：
- 成本較高
- 若使用過量，可能產生「黏滑感」
- 某些患者對黏膠質口感不適應

**適用情境**：長期使用、需要精確控制濃稠度、熱飲（茶、湯）使用、臨床建議優先使用。

## 第三部分：各IDDSI等級的使用比例

以下為一般參考比例，**實際用量依品牌不同而異**，請務必參閱各產品包裝說明。

### 以Resource ThickenUp（澱粉基）為例

| IDDSI等級 | 每150ml液體加入量 |
|---|---|
| L1（輕度稠） | 約1/2匙（1.5g） |
| L2（稍稠） | 約1匙（3g） |
| L3（中度稠） | 約1.5匙（4.5g） |
| L4（高度稠） | 約3匙（9g） |

### 以Resource ThickenUp Clear（黃原膠基）為例

| IDDSI等級 | 每150ml液體加入量 |
|---|---|
| L1 | 約1/4匙 |
| L2 | 約1/2匙 |
| L3 | 約3/4匙 |
| L4 | 約1匙 |

**操作提醒（黃原膠基特別注意）**：
1. 加入增稠劑後需充分攪拌或搖晃
2. **靜置3–5分鐘**讓黃原膠充分作用
3. 靜置後再次確認稠度，不足再少量補充
4. 切勿在等待期間持續加入更多——這是最常見的錯誤，導致過稠

## 第四部分：IDDSI濃稠度測試方法

### 湯匙測試（Spoon Test）

適用：L3–L4判斷

1. 舀一湯匙液體
2. 傾斜湯匙：L3應緩慢流下；L4應整體落下不再分離
3. 湯匙倒置：L4應留下明顯薄膜

### 注射筒測試（Syringe Flow Test，IDDSI官方方法）

適用：L1–L4精確測定（最準確）

1. 使用10ml注射筒（無針頭，藥局可購得）
2. 吸入液體至10ml刻度
3. 注射筒直立，開放出口
4. **10秒後**測量流出量：
   - L0（水）：>8ml
   - L1：4–8ml
   - L2：4–8ml（流速較L1略慢）
   - L3：1–4ml
   - L4：<1ml

## 第五部分：常見錯誤與解決方法

| 錯誤 | 後果 | 解決方法 |
|---|---|---|
| 不等待黃原膠作用直接加量 | 過稠、口感差 | 嚴格等待3–5分鐘後再評估 |
| 熱飲用澱粉基增稠劑 | 冷卻後過稠，熱時太稀 | 改用黃原膠基產品 |
| 不同批次使用不同品牌 | 每次稠度不同 | 固定一個品牌，記錄成功比例 |
| 以目視判斷稠度 | 不準確 | 使用注射筒流量測試 |
| 增稠後放置超過2小時（澱粉基） | 繼續增稠至不適合 | 澱粉基飲品現配現喝 |

## 第六部分：購買建議與費用估算

### 每月費用估算（以中度穩定需求患者為例）

- **澱粉基**（每天約6–8匙）：每罐可用約30天，月費NT$400–700
- **黃原膠基**（每天約3–5匙）：每罐可用約45天，月費NT$500–800

長照2.0輔助器具補助目前**尚不涵蓋增稠劑**，患者需自費。部分醫院社工可協助申請相關資源，建議諮詢。

---

## 結語

選擇增稠劑時，建議優先諮詢患者的語言治療師，確認合適的IDDSI等級後，再依預算選擇產品。若條件允許，黃原膠基增稠劑（如Resource ThickenUp Clear）在穩定性和安全性上均優於澱粉基產品，特別適合長期使用及熱飲需求的患者。

穩定的稠度，才能提供穩定的安全保護。
