# 台灣長照機構吞嚥障礙膳食管理——法規與實務

## 長服法膳食規範概覽

**長期照顧服務法（長服法）**於民國106年（2017年）全面施行，對住宿型長照機構的膳食服務訂有明確規範：

- **個別化飲食計畫**：機構應依住民的健康狀況、疾病需求及飲食偏好，制定個別飲食計畫，並定期複評
- **質地適當性**：住民如有吞嚥障礙，應提供符合其能力的適當質地飲食，並留有評估記錄
- **飲食記錄**：每餐進食情況應記載於照護記錄，異常狀況（嗆咳、拒食等）需通報護理人員
- **專業人員配置**：50床以上機構應設有專任或特約營養師；百床以上機構則需有護理師及物理治療師

違反規定者，主管機關（縣市政府社會局）可依長服法開罰並要求限期改善。

## 日照中心ABC供餐要求

長照2.0的**ABC服務體系**中，日間照顧中心（B單位之一）對供餐有以下要求：

**A單位（長照旗艦店）：**
- 整合資源，協助B單位媒合語言治療師進行吞嚥評估
- 建立轄區吞嚥障礙住民資料庫

**B單位（日照中心）：**
- 需提供午餐及點心，餐食應符合個別住民的質地需求
- 有吞嚥障礙住民需提供語言治療師認可的質地飲食
- 每季應安排一次吞嚥評估複評機制

**C單位（社區關懷據點）：**
- 供餐服務以共餐為主，吞嚥障礙個案應轉介B單位獲得更完整服務

## 語言治療師認定文件

長照機構在為吞嚥障礙住民調整飲食質地時，應保存以下文件：

1. **語言治療師吞嚥評估報告**：載明評估方法（床邊或儀器）、診斷結論及IDDSI建議等級
2. **飲食質地處方單**：包含固體食物等級（L3-L7）及液體濃稠度等級（L0-L4）
3. **複評記錄**：建議每3至6個月複評一次，或住民病情顯著改變時重新評估
4. **家屬知情同意書**：說明飲食限制的原因及潛在風險

上述文件在衛生主管機關或保險機構稽核時，為重要的照護品質佐證。

## 廚師吞嚥食訓練

機構廚師的培訓是確保吞嚥飲食品質的關鍵環節。台灣目前提供的培訓管道包含：

- **台灣吞嚥障礙專業學會（TSDSA）**：不定期舉辦機構廚師工作坊，涵蓋IDDSI理論與實作
- **衛生福利部護理及健康照護司**：將吞嚥飲食列入長照機構廚師進階訓練課程
- **台灣食品科技相關大學院所**：提供質地改良食品製作短期課程
- **在地農業改良場**：部分縣市與農改場合作，開發以在地食材製作的軟食食譜

廚師應能正確執行**叉子測試、湯匙測試及手指測試**以確認食物符合IDDSI標準，並了解食物在不同烹調條件下的質地變化。

## IDDSI與台灣傳統質地分級對照

台灣長照機構過去多採用**衛福部長照飲食質地分級（粗碎/細碎/泥狀）**，與IDDSI尚未完全接軌：

| IDDSI等級 | IDDSI名稱 | 台灣傳統對應描述 |
|---------|---------|--------------|
| L3 | 細碎及濕潤 | 細碎食 |
| L4 | 攪爛 | 泥狀食/攪打食 |
| L5 | 軟質及細碎 | 軟食/易咀嚼食 |
| L6 | 軟質及細碎（需咀嚼）| 普通軟食 |
| L1 | 稍濃稠 | 薄芡 |
| L2 | 低度濃稠 | 中芡 |
| L3 | 中度濃稠 | 濃芡 |
| L4 | 高度濃稠 | 布丁狀 |

TSDSA建議機構逐步導入IDDSI標準，以利跨機構轉介及國際資訊交流。

## SNQ認證與優質長照

**SNQ（Symbol of National Quality）國家品質標章**是台灣衛生福利類服務的重要認證：

- SNQ認證長照機構在評鑑項目中包含**膳食服務品質**評分
- 通過認證的機構需提供標準化的吞嚥飲食製作流程及訓練記錄
- 家屬選擇長照機構時，可查詢SNQ認證名單（www.snq.org.tw）作為品質參考

此外，縣市政府每年辦理的**長照機構評鑑**亦包含膳食服務項目，評鑑報告可至衛福部長照服務資訊平台（ltc.mohw.gov.tw）查詢。
