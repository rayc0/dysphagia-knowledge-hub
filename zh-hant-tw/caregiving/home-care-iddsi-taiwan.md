# 居家照護IDDSI飲食準備：家屬完整操作手冊

## 前言

當家人因中風、巴金森氏症或其他疾病而被語言治療師建議執行IDDSI特定飲食等級時，許多家屬面臨同一個挑戰：「我知道要吃L4（泥狀食），但我到底要怎麼做、做到什麼程度才算安全？」

本手冊將IDDSI飲食準備從概念轉化為可操作的日常步驟，適合無醫療背景的家屬或外籍照顧者使用。

---

## 第一部分：開始前的準備

### 確認患者的IDDSI等級

在開始準備飲食之前，必須先確認：

1. **患者需要的食物等級**（L3、L4、L5、L6）
2. **患者需要的飲料等級**（L0、L1、L2、L3）——食物和飲料等級通常不同
3. **是否有其他飲食限制**（低鈉、糖尿病飲食等）

這些資訊應來自語言治療師的書面指示。若沒有書面指示，請主動要求。

### 必備工具清單

**基本工具（必備）**：
- 手持式電動攪拌棒（immersion blender）——比桌上型攪拌機更方便、易清洗
- 細孔濾網（used for straining purée）
- 各尺寸量匙（measuring spoons）——不能用家用湯匙代替
- 食物溫度計
- 密封保存容器（玻璃或PP材質）
- 電子廚房秤

**進階工具（選配）**：
- 桌上型食物調理機（大量備餐時使用）
- 矽膠模具（方便冷凍分裝）
- 注射筒10ml（稠度測試用，藥局購得）

### 採購清單建議

**增稠劑**（依語言治療師建議選擇）：
- 黃原膠基優先（Resource ThickenUp Clear 或同類產品）
- 備用：澱粉基（適合預算有限時的短期使用）

**常備食材**（易於製作L4/L5的食材）：
- 嫩豆腐、雞蛋、去皮魚肉
- 根莖類蔬菜（地瓜、南瓜、胡蘿蔔、馬鈴薯）
- 香蕉、木瓜（天然接近L4的質地）
- 大骨高湯（自製或無添加市售版本）
- 燕麥片（易煮至極軟）

## 第二部分：各IDDSI等級食物製作指南

### L3（輕度稠食物 / Liquidised）

**質地特徵**：
- 可緩慢從匙中倒出
- 無任何顆粒或纖維
- 像稀布丁或濃稠優格

**製作要點**：
1. 將食物徹底煮熟至極軟
2. 以攪拌棒加入足量高湯攪打至完全均質
3. 過篩，去除所有纖維和顆粒
4. 調整液體量以達到L3稠度（以注射筒測試：10秒流出1–4ml）

**適合食材**：嫩豆腐、蒸魚、嫩雞胸肉、南瓜、地瓜

---

### L4（泥狀食物 / Puréed）

**質地特徵**：
- 光滑均質，無任何顆粒
- 可用湯匙舀起
- 盤子傾斜時不自行流動
- 落入盤中後不塌散

**製作要點**：
1. 食物煮至比L3更硬一些（但仍極軟）
2. 攪拌棒高速攪打至少2–3分鐘
3. **必須過篩** ——這是L4的關鍵步驟
4. 調整至正確稠度（傾斜盤子不流動）

**適合食材**：地瓜泥、南瓜泥、豆腐泥、雞蛋羹（茶碗蒸）

**常見問題**：如何讓L4保持形狀但不過硬？
- 加入少量馬鈴薯泥可改善質地
- 太稀加少許太白粉水（煮熟）
- 太稠加溫熱高湯

---

### L5（切碎食物 / Minced & Moist）

**質地特徵**：
- 食物切碎至4mm以下的小粒
- 軟濕，不乾燥
- 可用舌頭壓碎，不需牙齒咬斷
- 可堆成形狀

**製作要點**：
1. 將食物煮至非常柔軟（超過一般熟度）
2. 切碎至小粒（不可過大）
3. 與足量的醬汁、肉汁或高湯混合，確保滋潤
4. 確認無硬皮、果核或硬纖維

**適合食材**：嫩豆腐丁、豆乾切細（須煮軟）、去皮魚肉切碎、嫩蛋花

---

### L6（軟質食物 / Soft & Bite-Sized）

**質地特徵**：
- 食物切成1.5cm以下的塊狀
- 軟到可以只用舌頭和上顎壓碎
- 無堅硬、脆或黏的質地

**製作要點**：
1. 適當烹調（不過老）
2. 去除所有骨頭、硬筋、厚皮
3. 切成統一小塊，避免混入不同大小
4. 確保食物有足夠水分（不乾燥）

## 第三部分：批量備餐與保存策略

### 一週備餐規劃

**星期日批量烹調**：
- 準備3–4種基礎泥狀食物（地瓜泥、南瓜泥、雞肉泥、魚泥）
- 熬製大量高湯（骨頭湯、蔬菜湯）
- 分裝冷凍

**每日快速調製**：
- 從冷凍庫取出前一晚所需食材放冷藏解凍
- 解凍後加熱，加入高湯調整稠度
- 測試稠度和溫度後提供

### 冷凍保存方法

**使用冰塊模具**：
- 將L4泥狀食物倒入矽膠冰塊模具
- 冷凍至固體後，取出裝入密封袋
- 標明日期、食材、IDDSI等級
- 保存期：1–2個月

**解凍和加熱**：
- 前一晚放冷藏室慢速解凍（最安全）
- 緊急時可用微波爐低火解凍，再以水浴加熱
- 加熱後充分攪拌，測試中心溫度≥70°C（殺菌），再冷卻至55°C以下餵食
- 加熱後**必須重新測試稠度**——溫度影響稠度，特別是澱粉基食物

## 第四部分：每次進食前的安全檢查清單

### 五點安全確認（每次必做）

| 確認項目 | 標準 | 方法 |
|---|---|---|
| 1. 質地正確 | 符合指定IDDSI等級 | 注射筒測試或目視判斷 |
| 2. 溫度適中 | 40–55°C | 食物溫度計 |
| 3. 無異常顆粒 | L4以下完全均質 | 目視＋觸感確認 |
| 4. 患者準備好 | 清醒、坐直90度 | 確認姿勢和警覺狀態 |
| 5. 環境安靜 | 無電視或大聲音樂 | 確認環境干擾最小化 |

## 第五部分：台灣特色食物的IDDSI改造

### 台灣家常菜改造範例

| 原始食物 | IDDSI等級調整方向 |
|---|---|
| 滷肉飯 | L5：滷肉切極碎＋軟爛白飯；L4：整體攪打成泥（加滷汁） |
| 肉粥 | 天然L3–L4（確保無明顯米粒） |
| 茶碗蒸 | 天然L4（避免加入蝦仁等硬料） |
| 絲瓜蛤蜊湯 | L4：去殼蛤蜊，絲瓜攪打成泥，加入清湯 |
| 豆腐 | 嫩豆腐天然接近L4；板豆腐煮軟可達L5 |
| 水果 | 香蕉天然L4；木瓜天然L4；芒果攪打後L3–L4 |
| 地瓜 | 蒸熟後天然L5；加湯攪打可達L4 |

## 結語

居家IDDSI飲食準備需要一段學習期，但一旦建立了屬於自己的備餐系統，每日操作將會變得有規律。

最重要的原則是：**質地一致性**。每次給予患者的食物稠度應保持一致，這樣才能提供穩定的保護。

如有任何疑問，請隨時回到語言治療師處諮詢——這不是麻煩，而是對患者安全的保障。
