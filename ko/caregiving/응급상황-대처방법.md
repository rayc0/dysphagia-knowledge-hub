# 연하장애 응급상황 대처방법

연하장애 환자를 돌보는 모든 보호자는 응급상황에 대비해야 합니다. 질식(choking)이나 심각한 흡인이 발생했을 때 빠르고 정확한 대응이 환자의 생명을 구할 수 있습니다. 이 가이드는 홍콩에서 연하장애 환자를 돌보는 가족과 보호자를 위해 실용적인 응급 대처 방법을 제공합니다.

---

## 응급상황의 종류 이해하기

### 1. 질식 (Choking)

음식이나 이물질이 기도를 막아 호흡이 어려워지는 상태입니다.

**경도 질식 (부분 기도 폐쇄)**:
- 강한 기침이 가능함
- 말을 할 수 있음
- 호흡이 어렵지만 가능함

**처치**: 스스로 기침을 계속 하도록 격려. 등을 두드리거나 하임리히법 즉시 시행하지 않음 — 기침이 가장 효과적인 방법

**중증 질식 (완전 기도 폐쇄)**:
- 기침이 불가능하거나 매우 약함
- 말을 할 수 없음
- 호흡이 거의 없거나 전혀 없음
- 피부가 청색증(파란색)으로 변함
- 목을 두 손으로 감싸는 동작 (universal choking sign)

**처치**: 즉시 하임리히법 시행 및 999 신고

## 하임리히법 (Heimlich Manoeuvre) 단계별 지침

### 성인 환자 — 서 있거나 앉아 있는 경우

1. **위치 잡기**: 환자 뒤에 서기. 한 발을 환자의 두 발 사이에 넣어 균형 유지
2. **허리 찾기**: 환자의 허리 위치 확인
3. **주먹 만들기**: 한 손으로 주먹을 쥐고, 엄지 손가락 관절이 위를 향하도록
4. **위치 설정**: 주먹의 엄지 부분을 환자의 배꼽 위, 가슴뼈(흉골) 아래에 위치
5. **손 감싸기**: 다른 손으로 주먹을 감싸기
6. **압박**: 강하고 빠르게 위쪽, 안쪽으로 당기기 (J자 동작)
7. **반복**: 이물질이 나오거나 환자가 의식을 잃을 때까지 반복 (최대 5회)

**등 두드리기 (Back Blows)와 병행**:
- 하임리히법 5회와 등 두드리기 5회를 번갈아 시행하는 방법도 효과적
- 등 두드리기: 한 손으로 가슴을 지지하며 등 중앙을 손바닥 뿌리로 5회 강하게 두드리기

### 환자가 휠체어에 앉아 있거나 움직이기 힘든 경우

1. 환자 뒤에 무릎 꿇거나 서기
2. 위와 동일한 방법으로 하임리히법 시행
3. 필요시 환자를 바닥에 눕혀 **복부 압박법** 사용

### 환자가 의식을 잃은 경우

1. **즉시 999 신고** (또는 주변에 도움 요청)
2. 환자를 바닥에 눕히기 (단단하고 평평한 면)
3. **CPR 시작**: 가슴 압박 30회 + 인공호흡 2회 반복
4. CPR 전 입 안을 확인하여 이물질이 보이면 제거 (보이지 않으면 손가락으로 더듬지 않기)

### 임산부 또는 비만 환자

복부 압박이 어려운 경우: **흉부 압박법** 사용
- 가슴뼈 중앙 하부에 손을 위치하여 강하고 빠르게 뒤쪽으로 압박

## 홍콩 999 신고 방법

### 언제 999에 전화해야 하는가?

- 하임리히법을 3-4회 시행했는데도 이물질이 나오지 않는 경우
- 환자가 의식을 잃는 경우
- 환자가 청색증(피부, 입술이 파래짐)을 보이는 경우
- 호흡이 회복되지 않는 경우

### 999 신고 시 알려야 할 정보

999 안내원에게 명확하게 다음 정보를 전달하세요:

1. **위치**: 정확한 주소 (홍콩, 구룡, 신계 중 어디인지 + 도로명 + 건물명 + 층수)
2. **상황**: "연하장애 환자가 질식 중입니다" (Choking — dysphagia patient)
3. **환자 상태**: 의식 있음/없음, 호흡 있음/없음
4. **연락처**: 전화번호

**홍콩에서 중요한 사항**:
- 999는 영어와 광동어 모두 가능
- 한국어는 지원되지 않으므로 영어로 준비: "Emergency, choking patient, address is..."
- 전화를 끊지 말고 안내원의 지시를 따를 것

### 구급대원 도착 전 해야 할 일

- 계속 하임리히법 또는 CPR 시행
- 건물 입구로 누군가를 보내 구급대원을 안내
- 환자가 복용하는 약물 목록을 준비

## 흡인 증상 인식하기

흡인은 즉각적인 응급상황이 아닐 수 있지만, 반복되면 흡인성 폐렴으로 이어집니다.

### 급성 흡인 증상

| 증상 | 설명 |
|------|------|
| 기침 | 식사 중 또는 직후 갑작스럽고 강한 기침 |
| 젖은 목소리 | 식사 후 목소리가 젖은 듯 변하거나 쉰 소리 |
| 호흡 곤란 | 빠른 호흡 또는 호흡 노력 증가 |
| 청색증 | 입술이나 손끝이 파랗게 변함 |
| 공황 상태 | 호흡 불편으로 환자가 불안해하거나 공황 상태 |

### 만성/반복 흡인의 신호

- 식사 후 반복적인 기침
- 식사 시간이 점점 길어짐 (30분 이상)
- 식욕 감소, 음식 거부
- 반복적인 발열, 특히 식사 후
- 잦은 흉부 감염 또는 폐렴 진단
- 설명되지 않는 체중 감소

### 침묵성 흡인 (Silent Aspiration)

특히 위험한 것은 기침 없이 조용히 일어나는 흡인입니다. 노인, 뇌졸중 환자, 일부 신경계 질환 환자에서 발생할 수 있습니다.

**인식 방법**: 기침은 없지만 반복적인 폐렴, 원인 불명의 발열, 식욕 저하 등이 지속될 때 의심

## 예방적 조치

응급상황을 예방하는 것이 최선입니다.

### 식사 환경 안전 점검

- [ ] 적절한 식사 자세 확인 (최소 60도 이상)
- [ ] 처방된 IDDSI 단계에 맞는 음식/음료 준비
- [ ] 증점제 올바르게 사용
- [ ] 고위험 음식 제거 (견과류, 딱딱한 음식, 찰진 음식)
- [ ] 식사 속도 조절 (충분한 시간 확보)
- [ ] 식사 환경 정돈 (집중할 수 있는 환경)

### 구강 위생 유지

- 식사 전후 양치질 또는 구강 청결제 사용
- 입 안에 음식이 남아 있지 않은지 확인
- 의치(틀니)는 잘 맞는지 정기적으로 확인

### 보호자 교육

- **하임리히법 실습**: 글로만 배우지 말고 인체 모형이나 쿠션으로 직접 연습
- **CPR 교육**: 홍콩 적십자사 또는 St. John Ambulance에서 CPR 과정 수강 가능
- 응급 연락처 목록을 눈에 잘 보이는 곳에 붙여두기

### 응급 키트 준비

- 환자 의료 기록 요약본 (진단명, 복용 약물, 알레르기)
- 담당 언어치료사 연락처
- 담당 의사 연락처
- 가족 긴급 연락처

## 핵심 요약

1. **중증 질식** 시 즉시 하임리히법 시행 — 망설이지 않기
2. 의식 잃으면 **999 신고 + CPR** 시작
3. **흡인 증상**을 인식하고 담당 의료팀에 보고
4. **예방이 최선** — 올바른 자세, IDDSI 식이, 구강 위생
5. 모든 보호자는 **하임리히법과 CPR**을 실습으로 배워두기
